Base URL

Production console: https://console.mintmvp.com. Values below are saved in your browser only.

Where to get the project UUID

In the Console, open any project. The address bar looks like:
https://console.mintmvp.com/projects/<project-uuid>/overview
(you might see /tasks, /api, etc. instead of /overview).
Copy the first path segment after /projects/ — that 36-character hyphenated string is your project UUID. Paste it into “Project ID” below.

Resolved base URL

Tip: append ?origin=https://console.mintmvp.com&project=<uuid> to this page to prefill.

Authentication

Every request must send your secret key in the Authorization header:

Authorization: Bearer mint_your_api_key_here

Create keys in the app under Project → API → API Keys. Keys start with mint_ and are tied to one project.

Error codes

Code Meaning
200 Success (GET, PATCH, DELETE)
201 Created (POST)
400 Bad request (missing required fields or no updatable fields)
401 Unauthorized (invalid or missing API key)
403 Forbidden (key does not belong to this project)
404 Not found (item does not exist or belongs to another project)
500 Internal server error
